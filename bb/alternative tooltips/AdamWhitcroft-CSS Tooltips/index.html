CSS Tooltips

These CSS Tooltips can be applied to just about any element simply by adding a data attribute—namely data-tip—to your element.

You can apply the tooltip to inline and block level elements, giving you a great deal of flexibility in their application—some examples of which are below.

At the bottom of the page you will find a few useful coloured variants as well as a gravity variation.

Some usage examples

Applied to an anchor

Hover over any of the links in this paragraph aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. Excepteur sint voluptate cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id est laborum.

<a href="" data-tip="Yay tooltip!">voluptate</a>

Applied to a div

Hover over this div and you'll see a tooltip has been applied to the entire element. You can disregard the class—it's purely there to make the div stand out.

<div class="demo-div" data-tip="This is a div">Hover over this div…</div>

Applied to a paragraph

Hover over this paragraph and you'll see a tooltip has been applied to the entire p element.

<p data-tip="This is a paragraph">Hover over this paragraph…</p>

Applied to a blockquote

Hover over this blockquote and you'll see a tooltip has been applied to the entire blockquote element.

<blockquote data-tip="This is a blockquote">Hover over this blockquote…</blockquote>

Applied to a pre block

Hover over this pre block and you'll see a tooltip has been applied to the entire element. You'll also notice the tooltip has inherited the font-family of the pre block, so if you want to keep things consistent, consider setting a font-family to the tooltip.

Coloured tooltips

Help, success, alert & error

What if you'd like a help, success, alert or error tooltip?

<span class="help" data-tip="Need help?">help</span>
<span class="success" data-tip="Great success!">success</span>
<span class="alert" data-tip="Yellow alert!">alert</span>
<span class="error" data-tip="Oh noes!">error</span>

Pro tip: did you hover over the coloured words above?

Tooltip gravity

Above or below?

Tooltips don't always have to appear above your element, you can also set them to appear below.

This paragraph contains an anchor with a tooltip in the usual position. 

<a href="" data-tip="Above">anchor</a>

This paragraph contains an anchor with a tooltip below it. 

<a href="" class="tip-below" data-tip="Below">anchor</a>

This paragraph contains a success tooltip that appears beneath the element. 

<span class="success tip-below" data-tip="Even Greater Success!">success</span>

Caveat: Using the class="tip-bottom" won't work well in all situations. It is best applied to an inline elements such as a or span rather than a block level element.

Browser support

The :before and :after pseudo elements are supported in: Chrome, Firefox 3.5+, Safari 4+, Opera 6+, IE9+. Partial support in IE8.

The CSS rounded corners are supported in: Chrome, Firefox 4+, Safari 5+, Opera 6+ and IE9.
