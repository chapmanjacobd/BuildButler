Keyboard Controls

Key Action
W Pan Up
A Pan Left
S Pan Down
D Pan Right
Q, = Zoom In
E, - Zoom Out
R Reset Zoom
F Find
H Hide Part List (BOM)
Up Arrow, T Select Previous Part in List
Down Arrow, G Select Next Part in List
Left Arrow Select Previous Part Category
Right Arrow Select Next Part Category
Z Toggle Between Current Selected Part & Previous Selected Part
X Annotation mode (in development)
I Invert
O Set Overlay to Single Color
